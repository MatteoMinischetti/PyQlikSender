This is an example about how to export qlikview chart to excel and send it by email all with python.

How to use:
Make a new qlikview file qvw and name it PyQlikSender.qvw.
Import the qvs script.
Edit the script and substitute with your email data the inline table.
Reload the script.
Create two tables on the empty sheet:
emails and places.
save the qlikview file.

Edit the file PyQlikSenderConf.txt
set all the fields with your values.

run the python script, if everything is fine you will receive email with excel table attached.
